my portfolio

Military Movies

A One Page WordPress Site

I designed this first‐version WordPress site in collaboration with Military Movies, Inc. In order to build this site with a quick turnaround, I stripped down an existing theme, and created a child theme to style the front page as an HTML/CSS document, rather than dealing with plugins for the block editor. I focused plugins on speed optimization and SEO, utilizing Google Analytics.

Accelerate Marketing

Professional WordPress Exercise

This site is an exercise in locally editing an existing WordPress site downloaded from GitHub, using a child theme and staging site.
